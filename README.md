## **Вывод**

В ходе выполнения лабораторной работы №1 разработана программа на C++ для умножения квадратных матриц. Реализован классический алгоритм сложностью O(n³). Проведены эксперименты для матриц размером от 10 до 1000.

Полученные результаты:
- Для матрицы 1000×1000 время выполнения составило 0.83 секунды при 1 миллиарде операций.
- Все тесты пройдены успешно, результаты верифицированы через NumPy.
- Зависимость времени от размера матрицы соответствует теоретической O(n³).

Программа корректно читает матрицы из файлов, выполняет умножение и сохраняет результат. Автоматизированная система генерации, запуска и проверки работает стабильно.
